# iOSRaspiControl
### Clone
`git clone --recursive https://github.com/ph1ps/iOSRaspiControl.git`
##### This clones all submodules and the repository
### Install dependencies and start backend
```shell
cd ./NodeRoot
npm install
node index.js
```
